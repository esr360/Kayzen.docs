Welcome to Our User Guide

Overview

Welcome to the Kayzen user guide. Contained within your download should be a dev directory as well as a prod directory. This guide will focus only on the contents of the prod directory, which contains all the HTML, CSS and JavaScript files required to build your theme. The dev directory contains all the source files from which Kayzen is built. For a more comprehensive documentation focusing on the source files, view the Kayzen documentation on Github.

Compatibility

Whilst we thoroughly enjoy experimenting with the latest technologies, we also like to ensure that our products work on a wide vareity of devices. Our themes are compatible with all the latest browsers and devices, degrading gracefully where applicable.

Project ArchitectureAdvanced

Nexus is built using our own custom front end framework, One Nexus. Built using the popular HTML5 Boilerplate and Sass CSS extension language, Nexus is ready for the modern web out-the-box. With a unique and intuitive project architecture, customising your theme couldn't be easier.

If you are not comfortable using Sass you can still directly edit the compiled app.css stylesheet.

Basic HTML Structure

Each HTML page that is provided with this theme has a layout similar to the below example. This layout is based off the HTML5 Boilerplate and is ready for the modern web. Being built off the HTML5 Boilerplate means this theme comes with several useful advatages including:

  • Normalize.css for CSS normalizations and common bug fixes
  • jQuery via CDN, with a local fallback
  • Modernizr build for feature detection
  • Useful CSS Helper Classes
  • Default print CSS, performance optimized

Opera Compatibility Issues

Due to some compatibility issues with the Opera web browser, the following features have been modified:

  1. The Flyout Navigation is used on desktops instead of the Control Panel Navigation
  2. Some instances of dynamic circular images render as squares
